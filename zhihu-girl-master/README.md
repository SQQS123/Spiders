
- 本项目旨在抓取知乎上面妹子的头像，由于知乎用户URL不具有规则性，所以采用BFS算法搜索所有的用户。

- 本人在 Windows x64 的 python2.7.9版本下测试通过，其中需要的python库请大家自行安装。

- 本人部分采用了https://github.com/egrcc/zhihu-python 上的代码，感谢其无私分享。

- 配置： 在config中填写自己的邮箱和密码用于登录。
